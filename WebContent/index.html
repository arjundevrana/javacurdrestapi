Welcome to rest API demo.

Provide Content-type: application/json in all API calls.

  1. Find all country Get method
    http://localhost:8080/JAXRRestCurdApi/rest/countries
  2. Find all country in xml Get method
    http://localhost:8080/JAXRRestCurdApi/rest/countries/xml/
  3. Insert new country Post method
    http://localhost:8080/JAXRRestCurdApi/rest/countries
    Request parameter :-{"id":1(Int),"countryName":" England (string)","population":10000(int)}
  4. Update exist record Put method
    http://localhost:8080/JAXRRestCurdApi/rest/countries
    Request parameter :-{"id":1(Int),"countryName":" change country (string)","population":10000 change count(int)}
  5. Delete old record Delete method
    http://localhost:8080/JAXRRestCurdApi/rest/countries/4(record id)
